# SchoolManagement

Prerequisites
-------------

* [Java Development Kit (JDK) 8](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html)
* [Maven](https://maven.apache.org/download.cgi)

Build and Run
-------------
 
Run in the command line:
	```
	mvn package
	
	mvn spring-boot:run
	```
